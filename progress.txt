== Ralph Loop Iteration Log ==

--- Iteration 1: US-001 (2026-03-18) ---
Implemented: Restricted CORS from wildcard to https://wellvo.net
Files changed: edge-functions/server.ts
Patterns: ALLOWED_ORIGIN env var for configurability. Requests without Origin header (iOS, pg_cron) pass through. Disallowed origins get empty CORS headers on responses + 403 on preflight.
Gotcha: iOS native apps don't send Origin headers, so CORS must allow missing-origin requests.

--- Iteration 2: US-002 (2026-03-18) ---
Implemented: In-memory sliding window rate limiter per user+endpoint
Files changed: edge-functions/shared/rate-limiter.ts (new), edge-functions/server.ts
Patterns: Default 30 req/min. invite-receiver: 5/min, on-demand-checkin: 10/min. Service role bypasses. Returns 429 + Retry-After header.
Gotcha: Rate limiter is in-memory, so limits reset on container restart. Fine for single-instance Coolify deploy.

--- Iteration 3: US-003 (2026-03-18) ---
Implemented: Cloudflare Pages _headers file with CSP, X-Frame-Options, etc.
Files changed: website/public/_headers (new)
Patterns: Cloudflare Pages reads _headers from public/ dir. CSP needs 'unsafe-inline' for style-src due to Vite/React runtime styles. Google Fonts requires explicit font-src and style-src allowlist.
